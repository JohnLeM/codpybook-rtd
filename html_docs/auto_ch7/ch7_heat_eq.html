Problem statement

let’s consider the heat equation in a fixed geometry \(\Omega\) with null Dirichlet conditions:

\[\partial_t u(t,x) = \Delta u(t,x), \quad u(0,x)=u_0(x),\quad x \in \Omega, \quad u_{\partial \Omega}=0\]
To approximate this equation, we follow the following steps: Select a mesh \(X \in \mathbb{R}^{N_x,D}\) for the domain \(\Omega\). Pick up a kernel \(k\) generating a space of vanishing trace functions. We represent this equation as \(\frac{d}{dt} u(t) = \Delta_k u(t)\), with evolution operator \(u^{n+1} = B(\Delta_k,u^{n},dt,\theta)\) and \(\theta = 1\). This corresponds to the fully implicit case
