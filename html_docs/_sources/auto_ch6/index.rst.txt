Chapter 6: Applications to machine learning : supervised, unsupervised and generative methods
=============================================================================================

This gallery contains examples and tutorials for Chapter 6. The examples demonstrate how to use
the CodPy library for various machine learning and regression tasks.

Each example includes code and outputs, such as plots or results, to help you understand
how the library works.
